해당 프로그램은 윈도우즈 CUI 환경에 맞춰 제작되었으며 레지스트리 등의 시스템 변화는 일절 유발하지 않습니다.

일부 기능을 추가하거나 삭제하고 싶다면 아래 링크의 소스 코드를 복사하여 변경한 후 사용하시면 됩니다.

> https://github.com/stdFrog/WINAPI/


위 프로그램의 기본 기능은 다음과 같습니다.

- 터미널 등의 CUI 환경에서 텍스트로 된 달력을 화면에 출력합니다.
- 한가지 옵션을 가지고 있으며, 지정한 날짜의 달력을 불러올 수 있습니다.
- 아무런 옵션을 지정하지 않고 실행하면 현재 날짜의 달력을 화면에 출력합니다.
